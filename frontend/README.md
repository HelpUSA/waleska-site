
# Waleska Imóveis - Frontend (Next.js)

## Comandos

```bash
npm install       # instalar dependências
npm run dev       # rodar localmente (http://localhost:3000)
npm run build     # gerar build para produção
npm start         # rodar build de produção
```

## API

Definida em `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000
```
