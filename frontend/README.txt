# Waleska Imóveis - Frontend (Vite + React)

## Passos para instalar e gerar o build:

1. Instale as dependências:
    npm install

2. Rode em modo de desenvolvimento:
    npm run dev

3. Gere o build de produção:
    npm run build

A pasta `dist/` será gerada com os arquivos prontos para deploy (Vercel, Netlify, etc.)
